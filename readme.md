# Cinema

Projeto para a compra de ingressos em uma rede de cinemas, que exibe os filmes, aperitivos, sala e outras informações, com solicitação de pagamento tanto em pix ou cartão de crédito/débito.

### 📋 Pré-requisitos

```
IDE - Vscode
Nodejs
```
